## CEQL in {{page.heading}}

Use CEQL to query the following endpoints:

* GET /vendor-invoices
* GET /vouchers
* GET /invoices
* GET /{objectName}
* GET /{objectName}/{objectId}/{childObjectName}
* GET /orders
* GET /customers
* GET /accounts
* GET /employees
* POST /bulk/query
* GET /vendors
